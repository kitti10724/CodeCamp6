Differentiation

Tag ความเหมือน ความแตกต่าง
< hr> & < br> ไว้จัดรูปแบบของเว็บ

hr ใช้สร้างเส้นคั่น

br ใช้สําหรับการขึ้นบรรทัดใหม่
< ul>&< ol> คำสั่งแสดงผลรายการแบบมีลำดับ

ul การแสดงรายการแบบไม่มีเลขลำดับ

ol คือข้อมูลรายการแบบมีหมายเลข
< select> & < checkbox> & < radio button> ไว้สำหรับเลือกค่าที่ต้องการ

select เลือกค่าได้อย่างอย่างหนึ่งที่มีตัวเลือกหลายค่า เช่น จังหวัด

checkbox เลือกค่าได้หลายค่าพร้อมกัน เช้น ของที่ชอบ

radio เลือกค่าได้อย่างใดอย่างหนึ่ง เช่น เพศ
div & span กล่องข้อความ

div กล่องข้อความที่อยู่บรรทัดถัดไปจาก element ก่อนหน้า

span กล่องข้อความที่อยู่บรรทัดเดี่ยวกับ element ก่อนหน้า
script & style กำหนดค่าคุณสมบัติและรูปแบบของ tag ที่เรียกใช้

script กำหนดค่ารูปแบบที่แสดงผล

script กำหนดค่าคุณสมบัติที่ต้องการทำงานหรือประมวลผล
