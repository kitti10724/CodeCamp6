Input type

Tag การใช้งาน
text ข้อความ
checkbox ช่องสีเหลี่ยมให้คลิกเลือกข้อมูล
color แถบสีให้เลือกสี
date วันที่
datetime วันที่และเวลา
datetime-local กำหนด input value ให้เป็นที่ใส่ ข้อมูล local date/time
email กำหนด input value ให้เป็นที่ใส่ ข้อมูล Email Address ตั้งแต่ 1 Address ขึ้นไป
file
hidden
image
month ใส่ ข้อมูล เดือน
number ช่องใส่ค่าชนิดตัวเลข
password ช่องใส่ข้อมูลรหัสผ่าน
radio ช่องวงกลมไว้เลือกข้อมูล
range ที่ใส่ ข้อมุล ที่สามารถกำหนดช่วงของข้อมูลได้
reset ปุ่ม reset ค่า
search ช่องไว้กรอกค่าที่ต้องการค้นหา
submit เป็นการนำข้อมูลทั้งหมดในฟอร์ม โดยโปรแกรม เบราว์เซอร์ ส่งไปยัง เซิร์ฟเวอร์/td>
tel ที่ใส่ ข้อมูล หมายเลข โทรศัพท์
text ที่ใส่ ข้อมูลข้อความทั่วไป
time ใส่ ข้อมูล เวลา
url ใส่ ข้อมูล URL Address
week ใส่ ข้อมูล สัปดาห์
